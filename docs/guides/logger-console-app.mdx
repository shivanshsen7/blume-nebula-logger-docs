The Logger Console app provides a single point of access to Logger's core objects and reporting features for users with the appropriate permissions.

## Available Objects and Tabs

The console surfaces the following custom and standard objects:

- **Log__c** – Primary log records
- **LogEntry__c** – Individual log entries within a log
- **LogEntryTag__c** – Tags associated with specific log entries
- **LoggerTag__c** – Global logger tags
- **Reports** – Standard Salesforce reports tab
- **Dashboard** – Standard Salesforce dashboard tab
